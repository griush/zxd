# zxd - Zig Hex Dump
CLI tool to visualize the contents of binary files.

# Building from source
Last `zig` version tested is `0.14.0-dev.2643+fb43e91b2`.

Clone the repo and run `zig build`. You will find the executable in `zig-out/bin/`.
